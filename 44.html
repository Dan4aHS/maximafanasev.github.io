  9. добавление push, unhift, pop, shift
  10. some - удовлетворяет ли какой-либо элемент массива условию
  11. reduce - метод принимает функцию, которая имеет в качестве аргумента аккумулятор и значение
  12. every - метод проверяет, удовлетворяют ли все элементы массива условию
  13. map - метод принимает функцию в качестве параметра и создает новый массив с результатом вызова указанной функции для каждого элемента массива
  14. flat - метод принимает в качестве аргумента массив массивов и сглаживает вложенные массивы в массив верхнего уровня
  15. filter - метод принимает функцию в качестве параметра и возвращает новый массив, содержащий все элементы массива, для которого функция фильтрации передавалась в качестве аргумента, и возвращает ее со значением true.
  16. forEach - метод применяет функцию к каждому элементу массива.
  17. findIndex - метод принимает функцию в качестве параметра и в дальнейшем применяет ее к массиву. Он возвращает индекс найденного элемента, если элемент удовлетворяет условию проверяющей функции, переданной в качестве аргумента. Если не удовлетворяет, возвращается –1.
  18. find - метод принимает функцию в качестве аргумента и в дальнейшем применяет ее к массиву. Он возвращает значение элемента, найденного в массиве, если элемент удовлетворяет условию проверяющей функции. В противном случае оно возвращается со значением undefined.
  19. sort - метод принимает функцию в качестве параметра. Он сортирует элементы массива и возвращает их.
  20. concat - метод объединяет два или более массива/значения и возвращает новый массив.
  21. fill - метод заполняет все элементы массива одинаковым значением, от начального индекса (по умолчанию 0) до конечного индекса (по умолчанию array.length).
  22. includes -метод возвращает значение true, если массив содержит определенный элемент, и значение false — если нет.
  23. reverse - метод меняет порядок следования элементов в массиве на обратный. Первый элемент становится последним, а последний — первым.
  24. flatMap - метод применяет функцию к каждому элементу массива, а затем сглаживает результат в новый массив. Он объединяет метод flat() и метод map() в одну функцию.
  25. Map – это коллекция ключ/значение, запись, возврат, проверка ключа, удаление, очистка, размер коллекции, объекты как ключи
  26. Map из Object, Object из Map - Object.entries
  27. Object.fromEntries(array) на результате, чтобы преобразовать [] обратно в объект
  28. Set – это особый вид коллекции: «множество» значений (без ключей), где каждое значение может появляться только один раз.
  29. WeakMap – это Map-подобная коллекция, позволяющая использовать в качестве ключей только объекты, и автоматически удаляющая их вместе с соответствующими значениями, как только они становятся недостижимыми иными путями. WeakSet – это Set-подобная коллекция, которая хранит только объекты и удаляет их, как только они становятся недостижимыми иными путями
  30. [a, b, ...] - Деструктуризация позволяет разбивать объект или массив на переменные при присвоении.
  31. Остаточные параметры let [name1, name2, ...rest], Math.max(...arr), для слияния массивов, [...str] строка в массив символов
