12463 Откройте файл электронной таблицы, содержащей в каждой строке девять натуральных чисел. Определите количество строк таблицы, для чисел которых выполнены оба условия:

- в строке есть одно число, которое повторяется четыре раза, есть другое число, которое повторяется дважды, остальные три числа различны;

- среднее арифметическое трёх неповторяющихся чисел строки не меньше наибольшего из повторяющихся в строке чисел. 9_12463.txt 

k=0
for s in open('9_12463.txt'):
    a = sorted(map(int, s.split()))
    a4 = [int(x) for x in a if a.count(x)==4]
    a2 = [x for x in a if a.count(x) == 2]
    a1 = [x for x in a if a.count(x) == 1]
    if len(a4)==4 and len(a2) == 2 and len(a1) == 3 and \
       sum(a1)/3 >= max(a4[0], a2[0]):
        k+=1
print(k)
