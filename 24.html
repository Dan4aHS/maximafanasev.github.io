Текстовый файл состоит из A B C 8 9. Определите максимальное количество идущих подряд символов, среди которых ни одна буква не стоит рядом с буквой а цифра с цифрой

 c = s[i] + 'A' + s[i+1] + 'A' + s[i+2] + 'A'+ s[i+3]
