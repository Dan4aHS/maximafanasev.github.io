Вспомогательные методы
methods: {
	show: function() {
		let text = this.cape(this.text);
		alert(text);
	},
	cape: function(str) {
		return str[0].toUpperCase() + str.slice(1);
	}
}


Передача параметров в метод

Тогглинг элементов
toggle: function() {
	this.visible = !this.visible;
}
@click="toggle"
