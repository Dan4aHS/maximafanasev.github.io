  2. Если количество значащих цифр в четверичной записи числа чётное, то к этой записи в середину дописывается 0
  3. Если количество значащих цифр в четверичной записи числа нечётное, то запись не изменяется
  4. Например, для исходного числа 610 = 1024
  5. Укажите максимальное число N, после обработки которого с помощью этого алгоритма, получается число R, не большее, чем 180. В ответе запишите это число в десятичной системе счисления
def cc(x):
    s = ''
    while x>0:
        s = str(x%4) + s
        x //= 4
    return s

for n in range(1,100):
    n4 = cc(n)
    if len(n4) % 2 == 0:
        n4 = n4[:len(n4)//2] + '0' + n4[len(n4)//2:]
    r = int(n4)
    if r <= 180:
        print(n)
