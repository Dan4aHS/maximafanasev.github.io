Основы

  • строка, число
  • большое число, да нет

  • символ, объект

Взаимодействие:

  • окно информации
  • окно вопрос, ответ
  • окно подтверждения и ответ

виды преобразований

  • в строку
  • в число
  • в булевое

типы операций: + - * / % ! !!

Виды циклов:

  • for
  • for of
  • while
  • do while

== или ===

  • 1 == '1' // true
  • 1 === '1' // false

strict mode - убирает ошибки ранних версий js, можно весь файл, строку в функции, по умочанию в модулях

null не существует, undefined не найдено

& ||

  • & ищет первый false
  • || ищет первый true

типы таймеров

  • setTimeout - 1 раз
  • setInterval - постоянно (можно удалить таймер)

любой тип данных в булевый:

  • Boolean()
  • !!{}
  • '', 0, null, undefined, NaN, false

NaN? isNan()

!! Приведение к boolean

Обработка ошибок, "try..catch" работает только для ошибок, возникающих во время выполнения кода

Модули - обычно содержит класс или библиотеку с функциями

  • export function sayHi(user) {}
  • import {sayHi} from './sayHi.js';
  • В модулях всегда используется режим
  • собственную область видимости
  • Код в модуле выполняется только один раз
  • «this» не определён
  • являются отложенными (deferred)
  • Не допускаются «голые» модули
